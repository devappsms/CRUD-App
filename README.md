# CRUD-App

### Project 
This project is for simple CRUD opreations such as Read, Write and Delete. Also it has JWT token validation. 

### Running Project in local or Installation 

* Assuming you have installed  [npm](https://www.npmjs.com/package/npm) and [Node JS](https://nodejs.org/en/download/) 
* Clone the repository to your machine and go to the project root directory 
* Run below command 
    
```sh
npm install 
```
   
### Testing  
    
* library - chai and chai-http 
* To run test cases - Run below command 
        
```sh 
npm run test 
```
